# Demzo

Demzo is a modern messaging web app prototype with email/password login and step-up phone verification for new networks.

## Run locally

1. Install dependencies with npm install.
2. Start the dev server with npm run dev.

## Notes

This is a front-end prototype. It simulates trusted networks locally in the browser. A production implementation should move IP or device trust checks to the server before sending the verification challenge.
